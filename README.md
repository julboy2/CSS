# CSS

### position
.box1 .inner1 {<br />
opsition: static<br />
top : 30px <br />
left : 100px <br />
}<br />
- 기본값 static <br />
- relative : 기존값에서 top 30 , left 100 으로 이동  <br />
- absolute : 부모중 static 이 아닌 부모를 찾아서 top 30 , left 100 으로 이동  <br />
- fixed : 부모와 상관없이 top 30 , left 100 으로 이동 <br />
- sticky : 스크롤을 내릴때 top 30 , left 100 으로 고정되어 있다. <br />
